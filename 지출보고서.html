ℹ️
폐업 업체 안내

폐업 완료 업체는 공개 의무 소멸 — 지출보고서 신고 불필요. 단, 수정보완 기간(21~31일) 이전 폐업 완료 확인 필요. 폐업 시 판촉영업 신고증은 반납 해야 합니다. 휴업 업체는 폐업이 아니므로 제출 의무 유지.

1
심평원 접속 → KOPS 진입

2
회원가입
우측 상단 [회원가입] → "사업자등록번호 있는 판촉영업자(CSO)" 선택 → 기입하기 클릭 → 기본정보 입력 → 가입 완료 후 로그인
📌 1유형만 가입해도 모든 유형 서비스 이용 가능
3
인증서 등록 → 공동인증서 로그인
등록: 상단 [인증센터] → [국민, 기업/기관 → 인증서 등록] → 공동인증서(KICA 발급본) 등록 → 등록 완료 후 로그아웃
로그인: 로그아웃 상태에서 [로그인] → [공동인증서 로그인] 선택 → 등록된 인증서로 로그인
⚠️ 인증서 등록 전엔 공동인증서 로그인 불가
4
업체정보 작성 (Step 1)

📋 필수 입력 항목: 제출연도(2026년/회계연도 2025년) · 경제적이익 제공여부(있음/없음) · 영업구분/영업형태(판촉영업자) · 업체명·사업자등록번호 · 대표자명·생년월일 · 업허가·신고번호·일자 · 담당자정보(이름·연락처·이메일) · 첨부파일(허가·신고증 jpg/png/pdf 업로드)
모든 항목 입력 후 [저장 후 다음으로 이동] 클릭
5
경제적 이익 제공 여부에 따른 처리
경제적 이익 제공 여부 선택에 따라 다음 절차가 달라집니다.
❌ 경제적 이익 제공 없음
  • 경제적이익 제공여부 → [없음] 선택
  • 첨부파일 섹션에서 판촉영업 신고증 업로드 (.jpg/.png/.pdf)
  • 동시 운영 → [동시운영하지 않음] 선택
  • 저장 및 제출
✅ 경제적 이익 제공 있음
  • 경제적이익 제공여부 → [있음] 선택
  • 첨부파일·판촉영업 신고증 업로드
  • 동시 운영 → [동시운영함] 선택 후 영업구분·사업자번호 입력
  • 유형별 지출보고서 작성 진행
⚠️ 동시운영 기재 시 타 영업형태(의료기기 등)도 함께 입력 필수. 누락 시 제출 오류 발생.
6
유형별 지출보고서 작성 (이익 있는 경우)
지출보고서 작성 현황에서 유형 선택 → 해당 유형 작성
📌 제품설명회: 복수요양기관 / 개별요양기관 두 가지 유형, 드롭다운에서 선택 후 [작성하기] 클릭
입력 순서: 드롭다운 선택 → [작성하기] → 건별등록 팝업 → 의약품·요양기관 정보 입력(돋보기 검색) → 보건의료인 정보 → 지원금액 → 장소·일시 → [저장] → 목록에서 확인

📊 엑셀 일괄등록: [엑셀다운로드] → 양식 저장 → 번호·제품명(표준코드)·요양기관기호·성명·식음료비·장소·일시 입력 → [일괄등록] 클릭 → 작성한 엑셀 파일 업로드

🔍 의약품 조회: 제품명 검색창 옆 돋보기 클릭 → 검색 결과 좌측 목록에서 제품 선택 → 선택 시 우측으로 이동 확인 → [선택] 버튼 클릭하여 적용
🏥 요양기관 조회: 기관명 옆 돋보기 클릭 → 지역·종별·기관명으로 검색 → 목록에서 해당 기관 클릭하여 선택 → 요양기관기호 자동 입력 확인
7
최종확정 및 제출 완료
작성 완료 후 [최종확정] 클릭 → 비식별 처리 확인 → 확인함 체크 → [동의 및 최종확정] 클릭
✅ 제출 완료 후 확인 사항:
· 진행상태: 공개(최종제출) 확인 → 정상 제출 완료
· 경제적이익 제공여부: 있음/없음 선택값 확인
· 작성일 / 수정일: 제출 날짜 확인
· 업체정보 작성내역 / 지출보고서 작성내역: 클릭하여 입력 내용 재확인 가능

자주 묻는 질문

자주 문의되는 사항을 모았습니다

경제적 이익을 제공하지 않았는데도 제출해야 하나요?▼
네, 반드시 무실적 제출을 하셔야 합니다. [경제적이익 제공여부 → 없음 선택] 후 판촉영업 신고증을 업로드하고 제출하시면 됩니다.
공동인증서는 어디서 발급받나요?▼
한국정보인증(KICA) signgate.com에서 발급합니다. 보건복지전용(국민건강보험공단) 인증서를 선택하면 무료로 발급 가능합니다. 신규 발급은 서류를 지참하여 국민건강보험공단·국민연금공단을 방문해야 하며, 기존 진행자는 온라인 갱신이 가능합니다.
은행용 공동인증서(개인·범용)도 사용할 수 있나요?▼
사용 불가합니다. 건강보험공단에서 발급받은 인증서 및 전자세금계산서용 인증서만 인증됩니다. 일반 은행용 금융 인증서는 로그인이 되지 않습니다.
제품설명회 복수기관과 개별방문의 차이는 무엇인가요?▼
복수 기관 제품설명회: 국내 여러 요양기관 대상 개최. 교통비·기념품 5만원 이하·숙박·식음료 10만원 이하 지원 가능.

개별 방문 제품설명회: 개별 요양기관 방문 개최. 식음료 10만원 이하(1일), 판촉물 1만원 이하만 허용 (월 4회 이내).
엑셀로 일괄 등록도 가능한가요?▼
가능합니다. KOPS에서 [엑셀다운로드] 클릭 → 양식 저장 → 번호·제품명(표준코드)·요양기관기호·성명·지원금액·장소·일시 입력 → 저장 → [일괄등록] 클릭 → 파일 업로드 순으로 진행하시면 됩니다.
학술대회 지원금 지급 시점이 개최 시점과 다르면 어떻게 작성하나요?▼
원칙적으로 실제 지원금이 지급된 시점을 기준으로 작성합니다. 예시: '24.12.1. 개최 → '25.1.7. 지급 → 26년 지출보고서(2025 회계연도) 내역에 작성.
폐업 예정인 경우 지출보고서를 제출해야 하나요?▼
수정보완 기간(21~31일) 이전까지 폐업이 완료되면 지출보고서 공개 의무가 소멸됩니다. 단, 휴업 업체는 폐업이 아니므로 제출 의무가 있습니다.
제출 완료 확인은 어떻게 하나요?▼
지출보고서 작성 목록에서 진행상태가 "공개(최종제출)"로 표시되면 정상 접수 완료입니다. 제출 완료 시 담당자에게 카카오톡 또는 SMS로 안내가 발송됩니다.
